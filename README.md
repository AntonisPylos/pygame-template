## Getting Started

Install [pygame-cli](https://github.com/AntonisPylos/pygame-cli):
```bash
pip install pygame-cli
```

Clone this project:
```bash
pygame clone git@github.com:{REPO_OWNER}/{REPO_NAME}.git
```

Run the game:
```bash
pygame run {REPO_NAME}
```

Build executable:
```bash
pygame build {REPO_NAME}
```
